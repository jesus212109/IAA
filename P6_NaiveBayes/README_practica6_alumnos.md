# Práctica 6 - Material para alumnos

Este paquete contiene una versión base del código de la práctica sobre clasificación de texto y filtros anti-spam con Naïve Bayes.

## Ficheros incluidos

- `practica_6_naive_bayes_spam_base_alumno.py`: plantilla base para completar.
- `spam.csv`: dataset que se utilizará en la práctica.

## Qué debe hacer el alumno

1. Completar las zonas marcadas con `TODO`.
2. Ejecutar el script.
3. Analizar los resultados obtenidos.
4. Redactar un informe breve con interpretación de:
   - la representación Bag of Words,
   - el modelo Naïve Bayes,
   - el suavizado de Laplace,
   - la matriz de confusión,
   - las palabras más asociadas al spam,
   - y los mensajes de prueba inventados.

## Ejecución

```bash
python practica_6_naive_bayes_spam_base_alumno.py
```

pip install pandas scikit-learn matplotlib
